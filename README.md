# DigiTools

## Description
DigiTools is a React-based landing and product showcase app for premium digital tools.
It includes product browsing, a simple cart flow, and plan-based pricing sections in one clean interface.

## Technologies Used
- React.js
- Tailwind CSS, DaisyUI
- JavaScript (ES6+)
- React-Toastify 
- JSON

## Key Features
- See the products and add/remove items from cart with instant feedback.
- Toggle between Cart and Products, with live cart count and total price.
- View pricing plans with highlighted options and feature lists.
